# Simple React Boilerplate :rocket:

## About 

Simple react app template app that uses [create-react-app](https://reactjs.org/docs/create-a-new-react-app.html#create-react-app) but pre-configured to use [tailwind CSS](https://tailwindcss.com/) and [redux toolkit](https://redux-toolkit.js.org/).

## Prerequisites

This project requires NodeJS v12 or later and npm to run.
